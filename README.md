SmartSeat – 도서관 좌석 예약 웹서비스 (Prototype)
이 프로젝트는 대학 도서관 열람실 좌석을 보다 편리하게 이용할 수 있도록 만든 웹 기반 프로토타입입니다.
좌석 조회, 예약, 예약 내역 확인 등 기본적인 기능을 중심으로 구현했습니다.

1. 프로젝트 소개
SmartSeat는 도서관 열람실의 좌석 현황을 한눈에 확인하고, 원하는 좌석을 직접 예약할 수 있는 간단한 웹서비스입니다.
별도의 서버 없이 브라우저 저장소(LocalStorage)를 활용해 예약 정보를 관리하도록 구성했습니다.

2. 구현 기능
● 좌석 조회
열람실 선택 후 좌석 배치 확인

좌석 상태 표시(예약 가능 / 예약됨)

● 좌석 예약
좌석 선택 후 시간대 지정

예약 정보 저장(LocalStorage 기반)

● 마이페이지
내가 예약한 좌석 목록 확인

예약 취소 기능 제공

3. 기술 스택
HTML

CSS

JavaScript

GitHub Pages 배포

4. 실행 방법
GitHub Pages로 배포된 링크에 접속

열람실 선택 → 좌석 선택 → 시간대 선택

예약 완료 후 마이페이지에서 내역 확인 가능

5. 배포 링크
(여기에 GitHub Pages 주소 입력)

6. 참고 사항
본 프로젝트는 수업에서 작성한 PRD를 기반으로 제작한 프로토타입입니다.

실제 서비스가 아닌 데모 버전이며, 데이터는 브라우저에만 저장됩니다.
