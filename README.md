# SmartSeat – 도서관 좌석 예약 웹서비스 (Prototype)

이 프로젝트는 PRD를 기반으로 AI 코딩 에이전트를 활용해 제작한 웹서비스 프로토타입입니다.

## 📌 구현 기능
- 열람실 좌석 조회
- 좌석 상태 표시 (예약 가능/예약됨)
- 좌석 예약 기능 (LocalStorage 기반)
- 마이페이지에서 예약 내역 확인
